Tooltip Demo

I wrote this plugin as a lightweight js tooltip solution for my Rackmount Servers project. This tooltip works differently for both mobile and desktop versions. The code for this plugin is somewhat procedural and needs some updating. I plan on revisiting this project to update the code and complete a pure CSS solution to go along with it. To see the source code visit my github page at Tooltip.
